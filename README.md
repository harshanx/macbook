## ✦ Overview

A high-fidelity **MacBook M5** product showcase website built entirely with vanilla HTML, CSS, and JavaScript — no frameworks, no libraries. Inspired by Apple's cinematic product pages, featuring scroll-driven canvas animations, a bento grid breakthroughs section, and a premium dark UI.

---

## ✦ Features

| Feature | Description |
|---|---|
| 🎬 **Scroll Canvas Animation** | 240-frame hero laptop animation driven by scroll position |
| 🖼️ **Premium Enclosure Section** | Full-screen sticky scroll track with 153-frame animation |
| 🧩 **Bento Grid** | Chip showcase, Retina display, audio, battery, connectivity cards |
| 🤖 **Apple Intelligence Card** | Unified feature card with Writing Tools, Siri, Notifications & Privacy |
| 🌗 **Glass Morphism UI** | Backdrop blur cards with ambient glow effects |
| 🖱️ **Custom Cursor** | Interpolated cursor with glow trail |
| 📐 **Fully Responsive** | Mobile-first layout with adaptive grid |

## ✦ Getting Started

```bash
# Clone the repo
git clone https://github.com/harshanx/macbook-m5-showcase

# Open in browser — no build step needed
open index.html
```

> Works best in a modern Chromium-based browser. Serve locally for full asset loading (e.g. `npx serve .` or VS Code Live Server).

---

## ✦ Tech Stack

- **HTML5 Canvas API** — frame-by-frame scroll animation engine
- **CSS Custom Properties** — theming, transitions, glass effects
- **Intersection Observer API** — lazy animation triggers
- **RequestAnimationFrame** — smooth 60fps render loops
- **CSS Grid** — bento layout system
